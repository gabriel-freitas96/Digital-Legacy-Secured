# Herança Digital (CipherAegis)

Um aplicativo web seguro para armazenar memórias digitais, senhas e arquivos importantes, com proteção por senha e sistema de herdeiro.

## 🚀 Funcionalidades

### ✅ Implementadas
- **Sistema de Login/Cadastro** com validação de email e senha
- **Armazenamento Seguro** de memórias com criptografia básica
- **Upload de Mídia** (fotos e vídeos)
- **Proteção por Senha** para acessar memórias
- **Sistema de Herdeiro** para transmissão de dados
- **Edição/Exclusão** de memórias
- **Ordenação por Data** (mais recentes primeiro)
- **Feedback Visual** com mensagens de sucesso/erro
- **Loading States** em operações
- **Interface Responsiva** com design moderno
- **Parallax Background** com imagem personalizada

### 🔄 Melhorias Recentes
- Validação completa de formulários
- Botões de editar/excluir memórias
- Mensagens de feedback com animações
- Estados de loading
- Ordenação automática por data
- Melhor UX com hover effects

## 🛠️ Tecnologias

- **React 19** - Framework frontend
- **Vite** - Build tool e dev server
- **CSS3** - Estilização moderna
- **LocalStorage** - Persistência de dados
- **Base64** - Criptografia básica

## 📦 Instalação

```bash
# Clone o repositório
git clone <url-do-repo>

# Entre na pasta
cd heranca-digital

# Instale dependências
npm install

# Rode em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 🎯 Como Usar

1. **Acesse** `http://localhost:5173`
2. **Cadastre-se** ou use `admin`/`12345`
3. **Desbloqueie memórias** com senha `memoria123`
4. **Adicione memórias** com texto e/ou mídia
5. **Configure herdeiro** para transmissão
6. **Edite/exclua** memórias conforme necessário
